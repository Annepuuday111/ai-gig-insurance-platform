# AI Powered Insurance Platform for Gig Workers

## Problem
Delivery workers lose income when external disruptions like heavy rain, pollution, floods, or curfews stop them from working.

## Solution
We are building an AI-powered parametric insurance system that automatically compensates gig workers when disruptions occur.

## Target Users
Delivery workers from:
- Zomato
- Swiggy
- Amazon
- Zepto
- Blinkit

## Key Features
- AI-based risk prediction
- Weekly insurance pricing
- Automated claim triggers
- Fraud detection system
- Instant payout simulation

## System Workflow
1 Worker registers
2 AI calculates weekly premium
3 System monitors weather/traffic
4 Disruption detected
5 Claim automatically triggered
6 Instant payout sent to worker

## Tech Stack
Frontend: React  
Backend: Node.js / Express  
Database: MongoDB  
AI: Python / Scikit-learn  
APIs: Weather API, Pollution API

## Development Plan
Phase 1 – Ideation and architecture  
Phase 2 – Build insurance automation system  
Phase 3 – Fraud detection and payout system

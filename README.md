# AI Powered Insurance Platform for Gig Workers

## 1. Project Title
AI Powered Insurance Platform for Gig Workers

## 2. Problem Statement
Gig delivery workers lose income due to external disruptions such as heavy rain, floods, pollution, or curfews.

## 3. Target Users
- Zomato
- Swiggy
- Amazon
- Zepto
- Blinkit

## 4. Proposed Solution
An AI‑powered parametric insurance platform that automatically compensates workers when disruptions prevent them from working.

## 5. Key Features
- AI‑based risk prediction
- Weekly insurance pricing
- Automated claim triggers
- Fraud detection system
- Instant payout simulation

## 6. System Workflow
1. Worker registers
2. AI calculates weekly premium
3. System monitors weather/traffic
4. Disruption detected
5. Claim automatically triggered
6. Instant payout sent to worker

## 7. Weekly Pricing Model
| Plan     | Weekly Premium | Max Payout |
|----------|----------------|------------|
| Basic    | ₹20            | ₹300       |
| Standard | ₹40            | ₹600       |
| Premium  | ₹60            | ₹1000      |

## 8. Parametric Triggers
| Disruption | Condition          | Action          |
|------------|--------------------|-----------------|
| Heavy Rain | Rain > 60mm        | Auto payout     |
| Flood      | Flood alert        | Auto claim      |
| Pollution  | AQI > 400          | Compensation    |
| Curfew     | Zone lockdown      | Income payout   |

## 9. Tech Stack
- **Frontend:** React
- **Backend:** Node.js / Express
- **Database:** MongoDB
- **AI:** Python / Scikit‑learn
- **APIs:** Weather API, Pollution API

## 10. AI Integration
The AI model is used for:
- Risk prediction based on weather, traffic, pollution, and historical data
- Dynamic premium calculation
- Fraud detection using pattern analysis

Features fed to the model include rainfall, temperature, pollution levels, traffic congestion, and location history.

# Project requirements for AI Insurance platform

## System tools
- Node.js (v16+)
- npm (comes with Node)
- Maven (v3+)
- Java JDK 17+
- MySQL server (ensure it's running and accessible)

## Backend dependencies
The backend uses Maven to pull dependencies defined in `backend/pom.xml`.
To install them, run:

```bash
cd backend
mvn clean install
```

The key libraries include:
- spring-boot-starter-web
- spring-boot-starter-data-jpa
- spring-boot-starter-security
- mysql-connector-j
- io.jsonwebtoken:jjwt (for JWT)
- spring-boot-starter-validation

## Frontend dependencies
The frontend is a Vite/React project. Dependencies are in `frontend/package.json`.
Install them with:

```bash
cd frontend
npm install
```

Important packages:
- react, react-dom
- react-router-dom
- react-icons
- tailwindcss (with postcss and autoprefixer)
- vite

## Environment variables
Create a `.env.local` file in the frontend folder with:

```
VITE_API_URL=http://localhost:4000
```

The backend uses `src/main/resources/application.properties` to configure the
database connection (username `root`, password `Uday@2215302`, database
`ai_insurance`).

## Start commands
A simple shell script (`start.sh`) is provided at project root to launch both
services with one command.  See `start.sh` for details.  You can also start them
manually:

```bash
# backend
cd backend && mvn spring-boot:run

# frontend (in a separate terminal)
cd frontend && npm run dev
```
